Formål
Statisk HTML-version af varegrupper og kurv.

Siden åbnes direkte i browser via file:/// (lokal sti).
Der er ingen server og ingen Python i brug.

Denne version fungerer som referencepunkt for senere iterationer,
hvor logik og data gradvist kobles på.

Obligatorisk
- Forstå strukturen i dette HTML-dokument (head, body, tabel).
- Forstå at dette er en lokal, statisk fil, som tilgås direkte fra din computer.
- Kunne forklare forskellen på file:/// og http://.

Valgfrit
- Tilpas udseendet i css/styles.css.
- Udvid tabellen med flere varer.
- Ret tekster og overskrifter, så dokumentet fremstår konsistent.
